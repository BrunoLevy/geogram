◆ replay_latest_command()

static void GEO::SimpleApplication::replay_latest_command ( )
staticprotected

Executes once again the latest issued command.

SimpleApplication has a mechanism to handle user commands implemented in menus. The latest command is memorized, and not directly executed when the user invokes a menu, this is to avoid launching a command while in ImGui rendering/event handler. In addition, one can use this mechanism to replay the latest memorized command (by default, this is bound to the F5 key).
