Bundled third-party software

Geogram uses the following softwares:

  • AntTweakBar: Simple OpenGL GUI (by Philippe Decaudin) - License: zlib/libpng
  • Exact predicates (by Jonathan Shewchuk) - License: Public Domain
  • HLBFGS: Numerical optimization (developed by Yang Liu during his post-doc at Inria)
  • LM6: .mesh file format I/O (by Loic Marechal) - License: LGPL
  • RPLY: .ply file format I/O (by Diego Nehab) - License: MIT

Optionally, Geogram can be compiled with additional libraries (see "How to compile" )

  • TETGEN: mesh generation (by Hang Si) - License: Affero GPL (free for academic use, contact Hang Si for other uses)
  • TRIANGLE: mesh generation (by Jonathan Shewchuk) (free for academic use, contact Jonathan Shewchuk for other uses)
  • FPG: Predicate filter generator (by Andreas Meyer and Sylvain Pion)

Some test programs use the following libraries to validate the algorithms:

  • ANN: Kd Tree (by David Mount) - License: LGPL
